Justifications

HTML

HTML produced and validated on all pages so far. A majority of HTML done from scratch. Have dealt with a number of issues for example displaying text over and image, and working with complex carousel structures etc.

CSS

Used a combination of Bootstrap and our own CSS to apply styling to the pages. Have good level of confidence with bootstrap grids, containers, responsive breakpoints and many other bootstrap features

JS

Created files to handle animations of some features. Hover animations for Products page and sine waves (squigily lines) to partition sections. Also augumented code to deal with input spider and to handle removal of items from table on products page.

PNG

Images converted to png files, cropped, and colour corrected using Photoshop (which provided more features than GIMP). Also deleted image backgrounds to produced images of less conventional shapes as can be seen on home page.

SVG

Put a good amount of effort into creating simple but appealing SVGs produced images by freehand drawing over a template. Simplified paths of SVGs as well.

Server

Sends html files, js files and JSON objects to frontend. Manipulates database.

Databases

Database initialised server loads testimonials from it and sends to client side.

Dynamic pages

Header and footer loaded from server onto every page. Testimonials dynamically loaded onto testimonials page.
